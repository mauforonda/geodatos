Una lista actualizada de datos geográficos sobre Bolivia.

Estructura del repositorio:

- `descubrir/`: índice vivo de capas y monitoreo de fuentes geoserver.
- `evaluar/`: estimación, priorización y preparación de datasets candidatos a archivado.
- `archivar/`: preservación de datasets vectoriales y mantenimiento del índice archivado.
- `publicar/`: sitio estático y vistas públicas del índice vivo y del archivo.

[Explorar](https://mauforonda.github.io/geodatos/)
